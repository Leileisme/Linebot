## 二、使用說明


#### 【當季蔬果-每日預估價】

1. 輸入蔬果「品名」或「品種」
2. 每次使用一個單詞搜尋，例：香蕉、青江、蒜...
3. 馬上查詢「每周批發均價」、「每日預估零售價」、「是否符合行情」


#### 【傳統市場-休市日】
1. 輸入關鍵字「休市日、休市、休、營業時間、營業、時間、日期、市場」
2. 或按下方選單「休市日」(僅手機版)
3. 馬上知道近3個月的台北市場休市日

#### 【哪邊買的到?】
1. 輸入關鍵字「買、購、buy、哪裡、賣」
2. 或按下方選單「市場在哪」(僅手機版支援)
3. 即可查詢你附近的「傳統市場」或「量販店」

####  小提醒
1. 「一次多詞」或「空白」 , 會影響搜尋結果
2. 「換日」或「休市日」可能會查無資料
<br>
